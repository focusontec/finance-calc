## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Analytics:** Vercel Analytics + Google Analytics 4
- **SEO:** JSON-LD Schema, Open Graph, Twitter Cards, IndexNow

## Features

- 23 financial calculators
- Mobile responsive design
- SEO optimized with structured data
- AI crawler friendly (GPTBot, ClaudeBot, PerplexityBot)
- Automatic exchange rate updates (GitHub Actions)
- Fast static site generation

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for static export and deploys automatically to Vercel.

```bash
# Deploy to Vercel
npx vercel
```
